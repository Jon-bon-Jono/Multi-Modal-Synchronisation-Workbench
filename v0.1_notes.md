# syncwb map-nearest

## `MappingService.generate_nearest_mapping`

In cmd arg we dictate `mapping_version_id`. 
1. If this already exists, the client will refuse to overwite unless the user explicitly passes `--overwrite`.
2. If the user reuses an existing `mapping_version_id` and passes `--overwrite`, then the system should delete all rows for that mapping version (including SAMPLE_MAPPING rows) before inserting the regenerated mapping.

Writes to SYNC_MODEL, MAPPING_VERSION, SAMPLE_MAPPING

```
result = service.generate_nearest_mapping(
    source,                                                     #Has type TimelineSelection
    target,                                                     #Has type TimelineSelection
    mapping_version_id=args.mapping_version,                    #Custom string
    sync_model_id=args.sync_model_id,                           #None
    top_k=args.top_k,                                           #3 ?
    weak_support_threshold_ms=args.weak_support_threshold_ms,   #None
    max_allowed_delta_ms=args.max_allowed_delta_ms,             #None
    extrapolation_policy=args.extrapolation_policy,             #None
    allow_numeric_identity=args.allow_numeric_identity,         #None
)
```

`NearestMappingConfig` becomes: top_k=3, weak_support_threshold_ms=75.0, max_allowed_delta_ms=200.0, extrapolation_policy='disallow', allow_numeric_identity=False

`sync_id` uses source_run_id, source_device_type, source_timeline_model_id, target_run_id, target_device_type, target_timeline_model_id 
    (becomes `"sync_identity_"+f"{slugify(source_run_id)}_{slugify(source_device_type)}_{slugify(source_timeline_model_id)}__"+f"{slugify(target_run_id)}_{slugify(target_device_type)}_{slugify(target_timeline_model_id)}"`)

For each source (RGB) time, `_nearest_candidate_indices` finds closest candidate target (PC) times

We go through these candidates and classify the support `_support_labels` as either:
* "left_extrapolation", "outside_run"
* "right_extrapolation", "outside_run"
* "interpolation", "weak_support" (time delta is larger than weak_support_threshold_ms)
* "interpolation", "supported"
All candidates are appended to the SAMPLE_MAPPING table
* `is_primary` set to true if rank=1 (min time delta) and support is "supported" or "weak_support"
